# Experiment Launch Checklist

## Pre-launch
- [ ] Hypothesis + decision rule written before starting
- [ ] Metrics defined (primary + guardrails)
- [ ] Allocation set and justified
- [ ] Tracking/logging verified (dry run)
- [ ] Rollback plan defined
- [ ] Stakeholder alignment on interpretation (directional vs decision-grade)

## During run
- [ ] Monitor sample balance
- [ ] Monitor tracking health
- [ ] Monitor guardrails for large regressions
- [ ] Avoid ad-hoc stopping based on repeated peeking

## Post-run
- [ ] Report includes limitations (esp. rapid tests)
- [ ] Root-cause notes for surprising outcomes
- [ ] Next step defined (iterate / scale / powered test / stop)
