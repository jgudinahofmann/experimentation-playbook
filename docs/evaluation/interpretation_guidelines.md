# Interpretation Guidelines (Rapid vs Powered A/B)

## Rapid tests (timeboxed)
Use language like:
- “Directional signal within a 1–2 week timebox”
- “Not sized to detect small lifts reliably”
- “Recommend follow-up if promising”

Avoid:
- “Proved a small lift”
- “Guaranteed improvement”

## Powered A/B tests
Use language like:
- “Sized to detect an MDE with target power”
- “Decision-grade result for rollout”

Always include:
- what was tested
- duration and allocation
- primary metric + guardrails
- limitations and risks
- recommended next step
