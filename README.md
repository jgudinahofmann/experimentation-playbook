# Experimentation Playbook (Public-Safe)

This repo contains public-safe artifacts for running product experiments with rigor and speed:
- when to use rapid tests vs powered A/B tests
- evaluation + interpretation guardrails
- practical templates for planning, monitoring, and decisioning

> Disclaimer: generalized guidance only. No proprietary tooling or internal policies.

## Contents
- `docs/rapid-testing-vs-ab-testing.md` — why timeboxed rapid tests exist + tradeoffs
- `templates/rapid_test_onepager.md` — rapid test planning template
- `templates/ab_test_onepager.md` — powered A/B planning template
- `launch/experiment_launch_checklist.md` — monitoring + rollback checklist
- `evaluation/interpretation_guidelines.md` — how to report results responsibly
