# Experimentation Playbook (Public-Safe)

Practical guidance for running product experiments with speed and rigor:
- what **Rapid Testing** is and when to use it
- how it differs from powered A/B testing
- templates for planning, monitoring, and interpreting results responsibly

> Disclaimer: generalized guidance only. No proprietary tooling or internal policies.

## Contents
- `docs/rapid-testing.md` — definition, recommendation, decision tree, how-to
- `templates/rapid_test_onepager.md` — 1–2 week rapid test template
- `templates/ab_test_onepager.md` — powered A/B template
- `launch/experiment_launch_checklist.md` — monitoring + rollback checklist
- `evaluation/interpretation_guidelines.md` — how to report results
